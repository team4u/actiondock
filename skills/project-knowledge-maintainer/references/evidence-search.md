# Evidence Search

Search for evidence before writing substantive facts.

Useful evidence types:

- routes, controllers, handlers, middleware
- tests and fixtures
- database migrations, models, schemas, seed data
- package manifests and scripts
- config, CI, Docker, deployment files
- prompts, tools, agents, workflows
- existing docs and `.kb_inbox/` materials

For broad work, Scope Scanner should identify candidate domains before Domain Planner fan-out.
